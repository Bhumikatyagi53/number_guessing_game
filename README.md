# 🎯 Number Guessing Game (Python)

This is a simple command-line Number Guessing Game built using Python.

The computer randomly selects a number, and the player tries to guess it.
The game gives hints if the guess is too high or too low.

---

## 🚀 Features
- Random number generation
- User input handling
- Hints (higher / lower)
- Unlimited attempts (or limited if you coded that)
- Simple and beginner-friendly logic

---

## 🛠 Technologies Used
- Python
- random module
